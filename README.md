# Finora - Smart Expense Tracker

## Features
- User Registration and Login
- JWT Authentication
- Income Management
- Expense Management
- Budget Planner
- Savings Tracking
- Pie Chart and Bar Chart Analytics
- MongoDB Database Integration

## Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB
- Bootstrap
- Recharts

## How to Run

### Backend
cd server
node server.js

### Frontend
cd client
npm run dev
